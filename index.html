HTML

CSS

CSS 초기화 작업과 안정적인 CSS속성을 사용하여 크로스 브라우징(Cross Browsing)을 위한 적절한 배치와 디자인을 하고 있습니다.
디자인 시안의 구조에 따른 마크업과 스타일 입히기, CSS3로 다양한 효과와 움직임을 구현할 수 있습니다. 주석처리를 적절히 이용하여 코드를 보기 쉽게 정리할 수 있으며, 한 줄 코딩을 지양합니다.


JQUERY

코드의 작성 및 수정이 가능합니다.

상처를 내고 다시 일어나서 도전하고.. 그렇게 결국 온몸에 상처를 남기고서야 두 발 자전거를 탈 수 있었습니다.
몸에 남은 영광의 상처들이 자랑스러웠고, 해냈다는 성취감이 좋았습니다.

프로젝트를 완성해나가며 어린 시절 느꼈던 성취감을 다시 한번 느꼈고,
이제는 한 회사의 든든한 일원으로서 함께 나아가고 싶습니다.

맡은 역할은 책임감있게, 사고방식은 유연하게, 항상 열정을 다하는 웹 퍼블리셔 김영현입니다.
